# miden-counter

Experimental smart contract and counter implementation built with Rust.

## Overview

This project explores counter-based smart contract mechanics using Rust and decentralized execution concepts.

## Features

- Counter state management
- Increment and decrement logic
- Smart contract structure
- Rust-based implementation
- Experimental architecture

## Stack

- Rust
- Miden VM
- Git
- Web3

## Status

Work in progress.

## Goals

- Learn low-level smart contract development
- Explore decentralized execution
- Experiment with Rust architecture
